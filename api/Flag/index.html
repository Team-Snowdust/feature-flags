Flag

Represents an individual flag.

This provides convenience access to other library functions with this flag already provided. See FeatureFlags for more information about each function.

Properties​

name​

This item is read only and cannot be modified. Read Only
</>
Flag.name: string

The name of this flag.

create​

</>
Flag.create(config: PartialFlagConfig) → ()

Creates a new flag with the provided configuration.

caution

Receiving a Flag object generally indicates that a flag already exists. The create function should only be used to create flags that don't already exist.

See: create
